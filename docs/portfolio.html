Process

Build a web application or a single landing page has its details, it is needed to solve the problem first then you can code. Theres are steps I take for my apps.

  • Read the API documentation
  • Select each endpoint I'll use
  • Test endpoints in Postman
  • Do a wireframe in figma
  • Select a topic for the app
  • Search for colors related to the topic
  • Transform the wireframe to something prettier than just gray
  • Select technologies depending on requierements and size of project.
  • Start coding
